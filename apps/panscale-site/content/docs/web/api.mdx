# API

## createWebScaler

```typescript
function createWebScaler(element: HTMLElement, options: WebScalerOptions): WebScaler
```

Factory function that creates a Scaler instance with DOM event listeners attached.

`WebScalerOptions` extends `Partial<ScalerOptions>` with:

- `callback` — Transform update handler (required)
- `contentWidth` / `contentHeight` — Content dimensions (optional, defaults to element scroll size)
- `doubleTapZoom` — Target zoom level for double-tap (0 or omitted = disabled)
- `onResize` — Callback fired on element resize: `(size: { width: number; height: number }) => void`

All [ScalerOptions](/docs/core/api#scaleroptions) are accepted. Example without physics:

```typescript
const scaler = createWebScaler(element, {
  contentWidth: 1600,
  contentHeight: 900,
  callback: (values) => { /* ... */ },
  animating: false,
  bouncing: false,
});
```

## Event Adapters

- `attachMouseEvents(element, scaler)` — Mouse event adapter
- `attachTouchEvents(element, scaler)` — Touch event adapter
- `attachWheelEvents(element, scaler)` — Wheel event adapter
- `attachPointerEvents(element, scaler)` — Pointer Events API adapter
- `attachGestureEvents(element, scaler)` — Safari gesture events adapter
- `applyTouchAction(element)` — Sets `touch-action: none` on element
