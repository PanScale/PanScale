Install the packages:

```bash
pnpm add @scaler/core @scaler/react-native transformation-matrix
```

## Quick Start

```tsx
import { ScalerView } from "@scaler/react-native";

function App() {
  return (
    <ScalerView
      style={{ flex: 1 }}
      contentWidth={2000}
      contentHeight={1500}
      options={{ zooming: true, bouncing: true }}
    >
      {/* Your content here */}
    </ScalerView>
  );
}
```
