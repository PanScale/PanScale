Install the packages:

```bash
pnpm add @scaler/core @scaler/web @scaler/react transformation-matrix
```

## Quick Start

```tsx
import { ScalerView } from "@scaler/react";

function App() {
  return (
    <ScalerView style={{ width: "100%", height: 400 }} zooming bouncing>
      <div style={{ width: 2000, height: 1500 }}>
        <h1>Pan and zoom me!</h1>
      </div>
    </ScalerView>
  );
}
```
